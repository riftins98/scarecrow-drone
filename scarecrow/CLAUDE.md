# scarecrow

Python package for drone controllers, sensor interfaces, and navigation.

## Subdirectories
- `controllers/` — Flight control algorithms (wall follow, rotation, stabilization) (see `controllers/CLAUDE.md`)
- `sensors/` — Sensor abstractions (lidar) (see `sensors/CLAUDE.md`)
- `navigation/` — Future SLAM/localization (currently empty `__init__.py`)

## Files
- `__init__.py` — Package init
