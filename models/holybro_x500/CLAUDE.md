# holybro_x500

Holybro X500 V2 quadcopter model with the full sensor stack attached (optical flow, rangefinder, 2D lidar, mono camera).

## Files
- `model.sdf` — SDF definition: frame, rotors, and sensor plugin attachments (lidar_2d_v2, mono_cam, optical flow, rangefinder). This is the file to edit when moving/rotating sensors on the airframe.
- `model.config` — Gazebo model manifest (name, version, description).
