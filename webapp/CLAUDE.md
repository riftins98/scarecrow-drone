# webapp

Full-stack web application for flight monitoring and pigeon detection.

## Subdirectories
- `backend/` — FastAPI REST API server (see `backend/CLAUDE.md`)
- `frontend/` — React TypeScript UI (standard CRA structure)
- `output/` — Generated flight videos and detection frames (gitignored)

## Files
- `start.sh` — Launches backend + frontend together
- `Start Scarecrow Mac.sh` — macOS-specific launcher
