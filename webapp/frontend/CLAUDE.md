# frontend

React + TypeScript single-page app. Dev server on port 3000, proxies API calls to the backend on port 8000.

## Subdirectories
- `src/` ‚Äî App source (components, pages, services, types) (see `src/CLAUDE.md`).
- `public/` ‚Äî Static `index.html` bundled by CRA. Not a code directory.

## Files
- `package.json` ‚Äî npm deps (react, typescript, axios) and CRA scripts (`start`, `build`, `test`).
- `tsconfig.json` ‚Äî TypeScript compiler config.
