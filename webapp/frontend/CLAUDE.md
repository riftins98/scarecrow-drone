# frontend

React + TypeScript UI on port 3000. Created with Create React App; will eventually want a Vite migration but works as-is. All UI code lives in `src/`.

## Subdirectories
- `src/` — React app source (components, pages, services, types) (see `src/CLAUDE.md`)
- `public/` — Static assets bundled by CRA (favicon, index.html, manifest)
- `node_modules/` — npm dependencies (gitignored)

## Files
- `package.json` — npm dependencies + CRA scripts (`start`, `build`, `test`)
- `package-lock.json` — pinned dependency tree
- `tsconfig.json` — TypeScript compiler options
